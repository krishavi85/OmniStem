# Third-party software

OmniStem does not copy or redistribute the engine source code. Users install engines separately, and OmniStem invokes their documented command-line interfaces.

| Engine | Upstream repository | Code license observed upstream | Integration |
|---|---|---|---|
| Audio Separator | `nomadkaraoke/python-audio-separator` | MIT | External CLI adapter |
| Demucs | `facebookresearch/demucs` / maintained fork `adefossez/demucs` | MIT | External CLI adapter |
| Spleeter | `deezer/spleeter` | MIT | External CLI adapter |
| Open-Unmix | `sigsep/open-unmix-pytorch` | MIT | External CLI adapter |

Dependencies such as PyTorch, TensorFlow, ONNX Runtime, FFmpeg, NumPy, SoundFile, FastAPI, PySide6, Typer, Rich, and their transitive dependencies retain their own licenses.

This file is an attribution aid, not legal advice. Check the exact version installed in your environment.
